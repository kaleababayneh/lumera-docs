Try uploading a document to the Lumera blockchain. Once stored, you'll receive an Action ID that you can use to download the file at any time from decentralized storage.

<StorageApp />

## How it works

1. **Upload** — Select a file and submit it. The file is encoded using erasure coding and distributed across Supernodes on the Lumera network.
2. **Action ID** — After a successful upload, you receive a unique Action ID. Save this — it's the only way to retrieve your file.
3. **Download** — Enter your Action ID to reconstruct and download the original file from the network.

<Callout type="info">
  Files are stored permanently on-chain. Storage fees are paid in LUMERA tokens. See the [Quick Start](/docs/getting-started/quick-start) guide to learn how to integrate this into your own application.
</Callout>
