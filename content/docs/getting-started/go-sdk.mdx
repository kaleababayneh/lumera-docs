## Installation

```bash
go get github.com/LumeraProtocol/sdk-go
```

## Internal Architecture

The Go SDK natively powers the Supernode daemon and adheres strictly to the Cosmos SDK architectural guidelines. It isolates responsibilities into three distinct namespaces:

| Layer | Namespace Map | Execution Scope |
|---|---|---|
| **Control Plane (Consensus)** | `client.Blockchain` | Submits state transitions (`MsgRequestAction`, `MsgFinalizeAction`), simulates fees, and queries `x/supernode` configurations natively via standard Tendermint gRPC. |
| **Data Plane (Cascade DA)** | `client.Cascade` | Executes the 5-step upload lifecycle, handles LEP-1 encoding routines, and routes streams locally across the network. |
| **Data Mesh Gateway** | `client.Cascade` (Under-the-hood) | Evaluates Kademlia mesh topologies and binds to direct `SN-API` endpoints (`/api/v1/actions/cascade`). |

## SDK Instantiation

To bind the SDK, you must construct a `client.Config` struct detailing the `x/action` application paths and a local keyring.

```go
package main

import (
	"context"

	"github.com/cosmos/cosmos-sdk/crypto/keyring"
	lumerasdk "github.com/LumeraProtocol/sdk-go/client"
	"go.uber.org/zap"
)

func main() {
	ctx := context.Background()

    // 1. Initialize an ephemeral or file-backed keyring
	kr, _ := keyring.New("lumera", "test", "/tmp", nil)

    // 2. Map Tendermint and Cosmos bindings
	client, err := lumerasdk.New(ctx, lumerasdk.Config{
		ChainID:      "lumera-testnet-2",
		GRPCEndpoint: "grpc.testnet.lumera.io:443", // State query engine
		RPCEndpoint:  "https://rpc.testnet.lumera.io", // Block polling engine
		Address:      "lumera1...",
		KeyName:      "my-key",
	}, kr, lumerasdk.WithLogger(zap.NewExample()))
	
    if err != nil {
		panic(err)
	}
	defer client.Close()
}
```

## File Upload Mapping

The execution is fully abstracted. The `client.Cascade.Upload` namespace dynamically generates ADR-036 signatures, builds `MsgRequestAction`, blocks on Tendermint finality, and offloads byte arrays to the DA Mesh.

```go
res, err := client.Cascade.Upload(ctx, addr, nil, "./my-document.pdf")
if err != nil {
    panic(err)
}
fmt.Println("State Transition Finalized. Action ID:", res.ActionID)
```

## File Download Mapping

Retrieve deterministic shards via the permanent Action ID token.

```go
_, err := client.Cascade.Download(ctx, actionID, "./downloads/")
```

## Cryptographic Interfaces

### Multi-Key Multiplexing
The Go SDK supports a factory pattern for executing independent operations across horizontally scaled backend systems:

```go
factory, _ := lumerasdk.NewFactory(lumerasdk.Config{...}, kr)

// Derive specific signers contextually 
alice, _ := factory.WithSigner(ctx, aliceAddr, "alice")
bob, _ := factory.WithSigner(ctx, bobAddr, "bob")
```

The factory parses generic `secp256k1` (Cosmos Native) curves gracefully alongside generic EVM paths.

### Cross-Chain Composability (ICS-27)
Lumera natively executes interchain accounts via IBC. The Go SDK handles these abstractions securely via custom payload injection functions.

```go
res, err := cascadeClient.Upload(ctx, addr, nil, filePath,
    cascade.WithICACreatorAddress(icaAddr),
    cascade.WithAppPubkey(pubkey),
    cascade.WithICASendFunc(sendFunc),
)
```
