## `createLumeraClient`

Factory function that constructs a fully configured Lumera boundary client.

```ts
import { createLumeraClient } from "@lumera-protocol/sdk-js";

const client = await createLumeraClient({
  preset: "testnet",       // "testnet" | "mainnet"
  signer: wallet,          // OfflineSigner (Requires signArbitrary)
  address: "lumera1...",   // Sender address
  gasPrice: "0.025ulume",  // Gas price for fee estimation
});
```

**Parameters:**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `preset` | `"testnet" \| "mainnet"` | No | Canonical network routing. |
| `rpcEndpoint` | `string` | No | Custom RPC Tendermint endpoint. |
| `lcdEndpoint` | `string` | No | Custom Cosmos REST endpoint. |
| `snapiEndpoint` | `string` | No | Custom SN-API off-chain Gateway. |
| `signer` | `OfflineSigner` | No | Wallet signer (Requires ADR-036 compliance). |
| `address` | `string` | No | Sender address |
| `gasPrice` | `string` | No | Gas scalar (e.g., `"0.025ulume"`) |

**Returns:** `Promise<LumeraClient>`

## Data Plane Executions (`client.Cascade`)

The Cascade subsystem handles chunk derivations and direct network payloads.

### `uploader.uploadFile`

A macro-executor that natively runs file compression, BLAKE3 indexing, consensus registration (`MsgRequestAction`), and SN-API data streaming.

```ts
const result = await client.Cascade.uploader.uploadFile(fileBytes, {
  fileName: "document.pdf",
  isPublic: true,
  expirationTime: "1742553600", // Unix timestamp bound
  taskOptions: {
    pollInterval: 2000,         
    timeout: 300000,            
  },
});
```

**Parameters:**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `fileBytes` | `Uint8Array` | Yes | Uncompressed byte payload. |
| `options.fileName` | `string` | No | Base file-level metadata. |
| `options.isPublic` | `boolean` | No | P2P visibility constraint. |
| `options.expirationTime` | `string` | No | Node escrow deadline. |

**Returns:** `Promise<UploadResult>` containing `action_id` and `taskId`

---

### `uploader.prepareFile`

Derives deterministic IDs offline. Converts input to bytes and computes the BLAKE3 hash.

```ts
const { fileBytes, dataHash } = await client.Cascade.uploader.prepareFile(file);
```

**Returns:** `Promise<{ fileBytes: Uint8Array; dataHash: string }>`

---

### `uploader.registerAction`

Broadcasts the formal `MsgRequestAction` mutation to the consensus layer.

```ts
const { actionId, authSignature } = await client.Cascade.uploader.registerAction(
  { fileBytes, dataHash },
  options
);
```

**Returns:** `Promise<{ actionId: string; authSignature: string }>`

---

### `uploader.sendFileToSupernodes`

Streams raw byte payload directly to the SN-API Gateway via `HTTP POST`.

```ts
const uploadResult = await client.Cascade.uploader.sendFileToSupernodes(
  actionId,
  authSignature,
  fileBytes,
  { taskOptions: { pollInterval: 2000, timeout: 300000 } }
);
```

Retries up to 5 times for SN-API ingestion stabilization.

---

### `downloader.download`

Streams raw reconstruction blocks backwards from Kademlia storage natively to a JavaScript `ReadableStream`.

```ts
const stream: ReadableStream<Uint8Array> = await client.Cascade.downloader.download(actionId);
```

**Returns:** `Promise<ReadableStream<Uint8Array>>`

## Control Plane Queries (`client.Blockchain`)

Wrappers natively evaluating Tendermint parameters.

### `getActionParams`
Retrieves the maximum block and Supernode size constraints.

```ts
const params = await client.Blockchain.getActionParams();
// { max_raptor_q_symbols: number, ... }
```

### `getAction`
Queries the finality status (`PENDING` | `DONE`) of a registered execution index.

```ts
const action = await client.Blockchain.getAction(actionId);
// { creator, actionType, dataHash, state, ... }
```

### `getActionFee`
Dynamically evaluates standard node escrow dependencies in `ulume`.

```ts
const fee = await client.Blockchain.getActionFee(fileSizeBytes);
```

### `getSupernodes`
Queries active mesh density.

```ts
const nodes = await client.Blockchain.getSupernodes();
```

## Signer Abstractions

### `getKeplrSigner` / `getLeapSigner`

Automatically binds to browser extensions to derive deterministic key paths dynamically. Ensure network registration prior to hooking.

```ts
import { getKeplrSigner } from "@lumera-protocol/sdk-js";

const signer = await getKeplrSigner("lumera-testnet-2");
```

Functions inherently validate ADR-036 (`signArbitrary`) executions dynamically.

## Data Mesh Event Subscriptions

| Status Token | Category | Interpretation |
|--------|----------|---------|
| `sdk:completed` | Success | Lifecycle loop finished cleanly. |
| `sdk:upload_completed` | Success | P2P node confirmed distribution. |
| `sdk:download_completed` | Success | Streams terminated correctly. |
| `sdk:failed` | Terminal | Unknown fatal un-catchable error. |
| `sdk:supernodes_unavailable` | Terminal | Active set lacks DA mesh connectivity. |
| `sdk:registration_failure` | Terminal | `MsgRequestAction` consensus failure. |
| `sdk:processing_timeout` | Yield | Encoding SLA bounds breached. |
| `sdk:download_failure` | Terminal | Error re-building block arrays. |
